# Gusto Website — Landing Page

Simple one-page landing site for Gusto Pizzeria Ristorante (Janabiyah, Bahrain).

## Stack
Pure HTML/CSS — no framework, no build step. Open `index.html` directly in a browser.

## Files
- `index.html` — the entire page
- `images/hero.webp` — hero photo (copied from DEV-Gusto menu app)

## Brand system
- Fonts: Jost (wordmark), Cormorant Garamond (italic), Manrope (UI) — loaded from Google Fonts
- Colors: `#FFFFFF` bg · `#2D2828` ink · `#685A5A` text · `#C75D2C` accent (burnt orange)

## Buttons — update these hrefs when confirmed
- **Order Now** → `href="#order"` — replace with ordering platform URL or WhatsApp link
- **View Menu** → `href="#menu"` — replace with deployed menu app URL (e.g. `/Gusto/`)
